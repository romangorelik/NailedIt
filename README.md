# NailedIt

A full stack practice application focusing on Front End design building

## Technologies Used
- HTML5
- CSS3
- SCSS
- React
- Redux
- MongoDB
- C3.JS
- NodeJS
- Express
- Webpack
- Babel

## To run
1. Clone or download the project
2. npm install
3. npm run react-dev
4. npm run server-dev
5. npm run compile:sass
6. Open browser to http://localhost:4000/
